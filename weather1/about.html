About This Project

This dynamic weather dashboard was created to provide a clean, simple, and user-friendly way to get real-time weather information. It uses live data to display current conditions, hourly forecasts, and a 5-day outlook for any location in the world.

Core Features

  • Real-time weather data by location.
  • Detailed hourly and 5-day forecasts.
  • Interactive map to visualize locations.
  • Ability to save favorite cities for quick access.

Technology Used

This project is built with foundational web technologies and leverages powerful APIs to deliver its features.

  • HTML, CSS, and JavaScript for the core structure and functionality.
  • OpenWeatherMap API for providing comprehensive weather data.
  • Leaflet.js for the interactive map display.
  • Geolocation API to find the user's current location.
